4. Market Data Provider Chain
The backend tries providers in this order:
1. Coinbase spot price API
2. CoinMarketCap quote API, if COINMARKETCAP_API_KEY is configured
3. CoinGecko simple price API
4. Local fallback market data

5. Provider Behavior
- Coinbase is used first for simple current USD spot price.
- CoinMarketCap is retained as a richer optional provider when an API key exists.
- CoinGecko is used as an additional free fallback provider.
- If all live providers fail, backend returns fallback data.

6. API Response Metadata
Backend response includes:
- dataSource: "live" or "fallback"
- priceProvider: "coinbase", "coinmarketcap", "coingecko", or "fallback"
- lastUpdated timestamp

7. Current Price Formatting
- Large assets like BTC/ETH display rounded whole dollar prices.
- Mid-priced assets like AVAX/XRP/LTC display two decimals.
- Small-priced assets like DOGE display enough decimals to be useful.
