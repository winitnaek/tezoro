6. Premium Index Current State
Important Phase 4 note:
- Premium Index is NOT properly calculated yet.
- Current price is live.
- Premium Index values are currently fallback/static mock values.
- Backend now derives simpler premium descriptions from the existing premium value.
- For BTC, label is Coinbase Premium Index.
- For non-BTC assets, label is Exchange Premium Index.

Current interpretation logic:
- Positive premium: Coinbase/exchange pricing is stronger than global markets, suggesting stronger U.S. or exchange-side buying demand.
- Negative premium: Coinbase/exchange pricing is weaker than global markets, suggesting softer demand or selling pressure.
- Near zero: prices are aligned and buying/selling pressure is balanced.

Phase 4 should implement the real calculation:
Coinbase Premium % = ((Coinbase USD price - global average USD price) / global average USD price) * 100

For BTC, this can be called Coinbase Premium Index.
For ETH/AVAX/DOGE/XRP/LTC, use careful naming such as Exchange Premium Index or U.S. Exchange Premium unless the exact Coinbase-vs-global calculation is available.
