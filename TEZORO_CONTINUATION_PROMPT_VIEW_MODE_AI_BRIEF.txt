Completed Sprint 1: View Mode + Pro Analysis Premium Gate

     - overview
     - pro
   - Default mode is Overview.
   - Selected mode persists in localStorage.
   - Storage utility:
     - src/utils/viewModeStorage.js
   - If Pro Analysis was previously stored but premium gating becomes active, the app falls back to Overview.

2. Pro Analysis premium gate was added.
   - Uses existing Vite flag:
     VITE_PREMIUM_FEATURE_ENABLED
   - Interpretation:
     VITE_PREMIUM_FEATURE_ENABLED=true means premium gating is active.
   - For now, there is no paid-user state.
   - When gating is active:
     - Overview is available.
     - Pro Analysis is locked.
     - Clicking locked Pro Analysis opens the existing UpgradeModal.
     - Clicking locked Pro Analysis does not switch view mode.
   - No duplicate modal was created.
   - No Stripe/auth/subscription logic was added.

3. View Mode selector component was added.
   - File:
     - src/components/ViewModeSelector.jsx
   - Uses Reactstrap ButtonGroup style.
   - Renders:
     - Overview
     - Pro Analysis with lock icon when locked
   - Uses the same lock-icon pattern/class as premium crypto buttons:

   - Includes aria-pressed selected state.
