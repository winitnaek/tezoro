14. Premium Index / Exchange Premium
Backend file:
backend/src/services/premiumService.js

Real premium calculation is implemented when providers are available.

Formula:
premiumPercent = ((coinbasePrice - globalAveragePrice) / globalAveragePrice) * 100

Provider inputs:
- Coinbase spot price as Coinbase-side/exchange-side price.
- CoinGecko global/simple price as global reference price.

Labels:
- BTC: Coinbase Premium Index
- ETH/AVAX/DOGE/XRP/LTC: Exchange Premium Index

Premium source:
- premiumSource: live when calculated successfully
- premiumSource: fallback when calculation fails

Premium interpretation:
- Positive premium: exchange pricing is stronger than global markets, suggesting stronger buying demand.
- Negative premium: exchange pricing is weaker than global markets, suggesting softer demand or selling pressure.
- Near zero: exchange pricing is closely aligned with global markets, suggesting balanced demand.

Important:
- API does not break if Coinbase or global reference calls fail.
- If premium cannot be calculated, fallback premium data is used.
