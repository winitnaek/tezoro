Tezoro Sprint 1 Completed Work: View Mode + Pro Analysis Premium Gate

Context:
Tezoro is an existing React/Vite + Express crypto market dashboard app.
The Sprint 1 work added a dashboard View Mode selector and premium gating for Pro Analysis.
Keep the app JavaScript + JSX only. Do not add TypeScript, routing, auth, Stripe, MongoDB, or backend changes.

Completed Features:
1. Added dashboard View Mode support.
   - Default mode is Overview.
   - Supported modes are:
     - overview
     - pro
   - Selected view mode is persisted in localStorage.
   - If Pro Analysis was previously stored but premium gating is enabled, the app falls back to Overview.

2. Added premium gate behavior for Pro Analysis.
   - Uses the existing Vite flag:
     VITE_PREMIUM_FEATURE_ENABLED
   - Interpretation:
     VITE_PREMIUM_FEATURE_ENABLED=true means premium gating is active.
   - When gating is active, Pro Analysis is locked.
   - Clicking locked Pro Analysis opens the existing UpgradeModal.
   - Clicking locked Pro Analysis does not switch out of Overview.
   - No duplicate modal was created.

3. Added new frontend files.
   - src/components/ViewModeSelector.jsx
   - src/utils/viewModeStorage.js

4. Updated CryptoMarketDashboard.
   - Added viewMode state.
   - Added localStorage initialization.
   - Added Pro Analysis lock handler.
   - Reused existing UpgradeModal.
   - Overview mode renders:
     - Market Outlook Card
     - Current Snapshot
     - Price History Chart
     - AI Intelligence Center
     - Quick Takeaways
     - Disclaimer
   - Pro Analysis mode renders the full dashboard:
     - Market Outlook Card
     - Calculated Market Signal Card
     - Current Snapshot
     - Historical Performance
     - Market Structure
     - Technical Analysis
     - Technical Rating Summary
     - Key Price Zones
     - Price History Chart
     - AI Intelligence Center
     - Key Levels & Outlook
     - Performance Comparison
     - System Status
     - Quick Takeaways
     - Disclaimer

5. Updated header layout and styling.
   - Browser/desktop layout:
